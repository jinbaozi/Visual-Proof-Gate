# Visual Proof Gate Reports

Generated reports are written here when `npm run vp:gate` runs.

Required outputs:

- `design-intent.lock.md`
- `evidence.md`
- `responsive-matrix.md`
- `token-ledger.json`
- `asset-ledger.md`
- `state-matrix.md`
- `content-stress-report.md`
- `defect-backlog.md`
- `impeccable-routing.md`

The `screenshots/` directory stores viewport, theme, and state evidence.
